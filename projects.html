Deep Learning Projects
  • Project: Smart Contracts and Transaction Security
    View Project
    Description: The objective of this project was to predict whether an ether transaction is fraudulent and whether a smart contract is not safe. This project was a team project (of 2) for the Machine Learning Under an Optimization Lens course at MIT. A CNN was implemented for the smart contracts and an Optimal Classification Tree(OCT) was implemented for the Ether Transactional Data.
